Proje Açıklaması
Bu projede, siber güvenlik alanında Python ile geliştirilen etkili ve işlevsel bir program ortaya koydum.

Programı Çalıştırmak için

main.py dosyasını çalıştırmanız yeterlidir.
Program, kullanımı kolay bir arayüz sunar ve işlevlerini hızlı bir şekilde yerine getirir.
Dikkat Edilmesi Gerekenler

Bu programın çalışabilmesi için Scapy kütüphanesine sahip olmanız gerekmektedir.
Eğer Scapy yüklü değilse, aşağıdaki komutu kullanarak yükleyebilirsiniz:

* Terminale bu kodu yazarak Scapy kütüphanesini yükleyebilirsiniz: pip install scapy

Not: Python 3.6 veya üzeri bir sürüm kullanmanız tavsiye edilir.
